This is the slide deck for my talk on Property-Based Testing (PBT),
titled "From Examples to Exhaustive: Intro to Property-Based Testing".
It requires you to set up reveal.js, at the relative path
`reveal/dist/reveal.js` (though you can change that in the html),
with the notes, markdown, and highlight plugins.

At this time it does not require anything further,
but if I wind up doing it at more conferences,
I may make the versions automagically generatable
by using my "Genaver" tool, which you can find at:
https://github.com/CodosaurusLLC/genaver
